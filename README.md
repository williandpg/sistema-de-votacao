<h1><strong>Sistema de Votação</strong></h1>

<h2><strong>Descrição</strong></h2>
<p align="justify">
  Este projeto propõe o desenvolvimento de um sistema de votação eletrônico, capaz de cadastrar pessoas candidatas e pessoas eleitoras, conduzir o processo de votação e permitir a consulta dos resultados a qualquer momento.
</p>

<h2><strong>Funcionalidades</strong></h2>
<ul>
  <li align="justify"><strong>Cadastro de candidaturas</strong> com identificação única.</li>
  <li align="justify"><strong>Cadastro de eleitoras e eleitores</strong> com validações para evitar duplicidades.</li>
  <li align="justify"><strong>Registro de votos</strong> e controle para impedir múltiplos votos por pessoa.</li>
  <li align="justify"><strong>Apuração parcial ou final</strong> com exibição de resultados e contagem total.</li>
</ul>

  <h1><strong>Electronic Voting System</strong></h1>

  <h2><strong>Description</strong></h2>
  <p align="justify">
    This project is an electronic voting system that can register candidates and voters, coordinate the voting process, and allow checking the election results at any time.
  </p>

  <h2><strong>Features</strong></h2>
  <ul>
    <li align="justify"><strong>Candidate registration</strong> with unique identification.</li>
    <li align="justify"><strong>Voter registration</strong> with validations to prevent duplicates.</li>
    <li align="justify"><strong>Vote recording</strong> with safeguards against multiple votes per voter.</li>
    <li align="justify"><strong>Result counting</strong> and summarized output.</li>
  </ul>

  <h2><strong>Technologies Used</strong></h2>
